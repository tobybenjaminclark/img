# BNG Image Styler

A static GitHub Pages app for styling images in the browser. Users can upload an image, preview the original, generate a compressed output, apply asymmetric bezel corners, and download the result.

Default bezel values:

- Top left: `80px`
- Top right: `20px`
- Bottom left: `20px`
- Bottom right: `80px`

Everything runs client-side with HTML, CSS, JavaScript, and Canvas.
